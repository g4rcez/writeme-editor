# Product Guidelines - writeme

## Visual Identity
- **Minimalist Aesthetic:** Prioritize a clean, uncluttered interface. Use generous white space and subtle borders to define sections without visual noise.
- **Color Palette:** Maintain a restricted color palette, favoring neutral tones with a single, clear accent color for primary actions.
- **Typography:** Use clear, legible sans-serif fonts (like Assistant or Nunito, already in your fonts folder) for the interface, and monospaced fonts (like Cascadia or Comic Shanns) for code blocks.

## Voice and Tone
- **Professional & Concise:** Messaging should be direct and helpful. Avoid flowery language or excessive technical jargon unless necessary for context.
- **User Agency:** UI elements should clearly indicate their purpose, empowering users to find tools without intrusive onboarding.

## UX Principles
- **Distraction-Free:** The writing area should be the focal point. Secondary UI (toolbars, sidebars) should be easily collapsible or disappear during active writing.
- **Immediate Feedback:** Interactive elements like slash commands and rich previews (Math/Diagrams) should respond instantly to maintain the creative flow.
